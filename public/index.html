توثيق واجهة برمجة التطبيقات

المصادقة Authentication

POST /auth/register

تسجيل مستخدم جديد

المعلومات المطلوبة:
  • firstName - الاسم الأول
  • lastName - اسم العائلة
  • email - البريد الإلكتروني
  • password - كلمة المرور
  • confirmPassword - تأكيد كلمة المرور
  • phone - رقم الهاتف
  • province - المحافظة
  • role - نوع المستخدم (Student/Teacher)
  • academicStage - المرحلة الدراسية (للطلاب فقط)
  • subject - المادة الدراسية (للمعلمين فقط)
  • certificate - شهادة المعلم (للمعلمين فقط)
POST /auth/login

تسجيل الدخول

المعلومات المطلوبة:
  • email - البريد الإلكتروني
  • password - كلمة المرور
GET /auth/confirmEmail/:activationCode

تفعيل الحساب

يتم استخدام هذا الرابط لتفعيل حساب الطالب من خلال البريد الإلكتروني

PATCH /auth/forgetCode

طلب رمز إعادة تعيين كلمة المرور

المعلومات المطلوبة:
  • email - البريد الإلكتروني
PATCH /auth/resetPassword

إعادة تعيين كلمة المرور

المعلومات المطلوبة:
  • email - البريد الإلكتروني
  • forgetCode - رمز إعادة التعيين
  • password - كلمة المرور الجديدة
  • confirmPassword - تأكيد كلمة المرور الجديدة
